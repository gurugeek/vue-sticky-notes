## 實現工具

1. Vue
2. Vue-cli
3. Webpack
4. LocalStorage
5. Bootstrap4

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
